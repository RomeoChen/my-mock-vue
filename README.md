# 数据驱动模型
Vue的执行流程: 字符串模板 -> AST -> VNode -> 真正DOM
1. 获得模板
2. 生成抽象语法树 AST
3. 利用Vue构造函数中提供的数据来替换模板中的数据，得到新的元素
4. 将新的元素替换页面中的模板
其中最消耗性能的是：模板 -> AST 这一阶段
